Acquiring Tools

Acquiring tools in minecraft is the first thing you need to do when starting a new world. Start by punching down a tree and crafting wooden tools. After getting wooden tools it's recommended to go for atleast a stone pickaxe since it can break iron which can then be smelted and used to make even stronger tools.
